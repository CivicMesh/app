# CivicMesh Mobile App

**CivicMesh** is a community-driven mobile platform that connects neighbors during emergencies and everyday challenges. Built with React Native and Expo, the app enables real-time reporting, response coordination, and resource sharing within local communities.

## What is CivicMesh?

CivicMesh transforms how communities respond to urgent situations by creating a hyperlocal network where residents can:

- **Report emergencies** like fires, floods, accidents, or security threats with precise geolocation
- **Request help** for immediate needs (medical assistance, shelter, supplies)
- **Share resources** such as food banks, charging stations, medical facilities, or temporary shelters
- **Coordinate responses** by marking "On My Way" to help neighbors in need
- **Resolve situations** with photo documentation and status updates

### How It Works

1. **Sign up** with your email and create an account
2. **Browse** the map or feed to see what's happening nearby
3. **Post** when you need help or spot an urgent situation
4. **Respond** to neighbors' posts by marking "On My Way"
5. **Resolve** posts when situations are handled, with photo proof
